# Installation

Install ags for your distro - https://aylur.github.io/ags/guide/install.html

clone the repository

```
git clone https://github.com/zenzawa/needy-widgets.git
```

cd into the respective widget directory

run `ags run app.ts`

## for nixos

clone the repository

```
git clone https://github.com/zenzawa/needy-widgets.git
```

cd into the respective widget directory

run `nix develop`

then `ags run app.ts`


# About widgets

## Ame

- It displays ame/kAngel animations. 
- The animations switch every 15 seconds.
- Click on the widget to switch between ame and kAngel

## Depaz

- click on the widget to consume meds

## SNS

- shows ame/kAngel images
- changes the images every 2 mins
- click on the widget to change the image
